Carousel CK is a responsive slideshow plugin that shows your images with a nice carousel effect.

== Description ==

Carousel CK can display your images or videos with nice effects. It is compatible with mobiles, responsive design (adaptive width) and you can slide with your fingers on your mobile. You can easily manage the slides with a Drag & Drop interface.

== Installation ==

1. Unzip the package `carousel-ck.zip`
2. Upload the folder `carousel-ck` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
